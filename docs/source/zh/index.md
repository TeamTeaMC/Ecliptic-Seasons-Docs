## 1.什么是节气？

正如其名字一样，这个模组为了Minecraft带来了新的季节体验，也可以叫它Solar Terms。
这是一种古老的中国历法，它将一年分为四季二十四段时期，每一段节气都有其独特之处。
知晓时候，顺应时节，可以达到人与自然的和谐，所谓“天人合一”。

## 2. Ecliptic Seasons的功能是否与静谧四季等模组重复了，或者是否可以一起工作。

Ecliptic Seasons是一个全新且独立的季节模组。
我们的目标不是替代现有模组或重复已有功能，而是因为Minecraft已经发展了十多年，我们希望带来更深入、更丰富的季节体验。
不仅仅是改变群系颜色，而是让更多植物发生变化；不仅仅是冬季的落雪，而是让整个世界的方块外观随季节演变。
不仅关注季节限制，还让作物在适宜湿度下茁壮成长。
甚至加入更多视觉和听觉上的细节点缀，提供沉浸式的季节氛围。
同时我们也会尽可能优化性能体验，欢迎大家积极反馈。

## 3. Ecliptic Seasons还能让我获得什么？

* 随节气改变的昼夜长短变化，冬夜漫漫夏日长。
* 基于节气的降雨系统，春日阴雨绵绵，秋日秋高气爽。
* 兼顾季节温度与群系湿润度的农业生长，以及一个简单的新温室。
* 动物繁殖需求随季节变化（自行启用）。
* 四季之声与群系颜色，甚至是方块模型，感受时光在指尖流转。
* 一些视觉上的额外点缀，如夏日傍晚低矮花草中飞出的萤火虫或者是树叶的落叶飘飞。
* 以上这些功能都可以通过配置文件或者数据、资源包进行配置，选择权在您！
* 如果您是服务器管理者，可以用`/ecliptic`来设置时间或者是天气。
* 如果您是整合包制作者或者其他模组制作者，您可以通过API获取当前状态，或者是制作数据包，更多自定义内容正在路上。
