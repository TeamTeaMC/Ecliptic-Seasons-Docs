# 节气 快速开始

欢迎来到 节气。

节气 为 Minecraft 引入了基于二十四节气的季节系统。随着时间推移，世界会逐渐发生变化，包括昼夜长度、天气、植被颜色、农业环境以及冬季景观等内容。

本指南将帮助你快速了解最重要的机制。

------

# 节气与季节

游戏中的一年由二十四个节气组成。

随着节气变化，你会观察到：

- 白天和夜晚长度变化
- 气候逐渐转暖或转冷
- 部分群系开始降雪
- 植被颜色发生变化
- 季节环境粒子与音效变化

当进入新的节气时，聊天栏会显示对应提示。

------

# 生长检测计

如果你准备开始农业玩法，建议优先制作：

**Growth Detector（生长检测计）**

右键作物后可以查看：

- 当前生长概率
- 温室等级
- 条件是否满足

当作物生长缓慢时，它通常是最直接的诊断工具。

------

# 季节农业

启用季节农业后，不同作物在不同季节会具有不同的生长效率。

这并不意味着作物完全无法生长，而是自然生长速度会受到影响。

默认情况下：

- 季节会影响生长概率
- 湿度会影响生长概率
- 骨粉仍然有效

因此即使环境不理想，你通常仍然能够完成种植。

------

# 温室

如果你希望获得更稳定的农业环境，可以建造温室。

温室本质上是一个封闭空间。

成功识别后，温室能够：

- 改善农业环境
- 减轻季节影响
- 配合湿度系统提升生长条件

对于长期农场而言，温室通常是最有效的解决方案。

------

# 湿度调节

部分作物对湿度环境较为敏感。

你可以使用以下设备调整温室湿度：

## 恒湿水箱

提高附近区域湿度。

适用于干燥环境或加湿需求较高的作物。

## 温室除湿器

降低附近区域湿度。

适用于过于潮湿的环境。

如果不确定当前环境是否合适，请使用湿度计查看实际湿度。

------

# 季节核心

季节核心是温室系统的重要组件。

如果你希望进行反季节种植，则需要在温室内放置对应的季节核心来改善季节条件。

------

# 如何获得季节核心？

季节核心 主要有以下获取方式：

- 完成相关成就奖励
- 进行 Seasonal Prayer Ritual（季节祈祷仪式）
- 偶然情况下从流浪商人处购买

对于普通种植而言并非必须，但反季节农业通常离不开它。

------

# 冬季与覆雪

冬季到来后，世界会逐渐出现覆雪景观。

默认情况下：

- 覆雪主要是视觉效果
- 不会修改世界方块
- 不会生成大量真实雪层

因此能够获得更好的性能和兼容性。

如果你更喜欢传统雪层表现，也可以在配置中启用原版方块雪机制。

------

# 中暑

在炎热群系中，夏季正午长时间暴露在阳光下可能触发 Heatstroke（中暑）。

你可以：

- 前往阴凉区域
- 避免长时间暴晒
- 佩戴具有抗火效果的头盔（更多可以查看数据包基础一章）
- 在配置中关闭该功能

------

# 常见问题

## 作物为什么生长缓慢？

请使用 生长检测计 检查：

- 当前季节
- 环境湿度
- 温室等级

------

## 骨粉为什么仍然有效？

这是默认行为。

节气 默认不会完全禁止骨粉催熟。季节和湿度主要影响自然生长概率。

------

## 反季节种植需要什么？

通常需要：

- 温室
- 季节核心
- 合适的湿度环境

------

## 覆雪效果与资源包冲突怎么办？

部分资源包或连接纹理模组可能会修改方块模型，导致覆雪显示异常。

你可以：

- 关闭覆雪渲染
- 使用经典覆雪资源包（1.20.1）
- 提交兼容性 Issue
